## 📈 진행
- 자신이 공부한 내용의 코드 및 정리한 내용을 자신의 브런치에 올립니다.
- 자신이 공부한 내용을 main브런치에 Pull Request를 요청합니다.
- 월요일마다 완료한 사람에 ✔️넣습니다.

## 🔲 **스터디 진행 방식**
#### Branch
- `barnach - 사용자명`
#### Commmit Message
- `자신이 공부한내용`
#### 파일 및 폴더 구조
- `챕터/이름`
- `Chapter02 변수(Variable)/원진.md`
