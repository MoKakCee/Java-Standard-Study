# Chapter 2 변수(Variable)

## 변수의 선언과 저장

### 변수

- 데이터(data)를 저장하기 위해 프로그램에 의해 이름을 할당받은 메모리 공간

### **변수의 선언과 초기화**

- 변수는 값을 저장할 공간을 마련하기위해서 사용한다
- 변수의 선언 방법 : 변수타입 변수이름
- 변수를 사용하기전 처음으로 값을 저장하는것을 `변수의 초기화` 라고한다.

```java
//변수의 선언 방법 : 변수타입 변수이름
int age; // 정수(int)타입의 변수 age를 선언
age=25;  // 변수 age에 25를 저장
int x=0; //변수 x를 선언 후, 0으로 초기화
int y=5; //변수 y를 선언 후, 5으로 초기화
int x=0, y=5 // 변수 타입이 같다면 한번에 선언과 초기화가 가능하다

```

## 상수와 리터럴

### 상수

- `값을 저장할 수 있는 공간` 이지만, 변수와 달리 한번 값을 저장하면 `다른 값으로 변경할 수 없다`.

```java
final int a; //정수형 상수 a를 선언
a=10;        // 상수에 처음으로 값 저장
a=20;        // 상수에 저장된 값을 변경할 수 없음
```

### 리터럴

- 그 자체로 값을 의미한다

```java
int a = 2022;     //a는 변수 2022는 리터럴
final int b = 10; //b는 상수 10은 리터럴 
```

## 두 변수의 값 변경

### 두 변수 x와 y에 저장된 값바꾸기

```java
int x=10, y=20; //x와 y에 리터럴 대입 
int temp=x;     //x를 저장할 공간 temp를 만든
int x=y;        //x에 y값을 넣음
int y=temp;     //y에 x의 리터럴을 저장했던temp를 대입하면 변경 완료
```

## 기본형과 참조형

### 기본형

- 논리형(boolean), 문자형(Char), 정수형(int, long, byte, short), 실수형(float, double)
- 계산을 위해 실제 값을 저장한다. 모두 8개

### 참조형

- 객체의 주소를 저장한다. 8개의 기본형을 제외한 나머지 타입.

## 정수형의 오버플로우

- 연산과정에서 타입이 표현할 수 있는 값의 범위를 넘어가는 상태를 말한다.
- 에러가 발생하지는 않지만 예상했던 결과 값이 아닌 전혀 예상하지 못한 값을 얻게되므로 오버플로우가 발생하지 않도록 적절한 크기의 타입을 고려해서 선택하고 사용해야한다.

```java
byte num1 = 127; // byte타입의 오버플로우 예시 (최대값 → 최소값)
byte num2 = -128; // byte타입의 오버플로우 예시 (최소값 → 최대값)

num1++; // 127 + 1
num2--; // -128 - 1

System.out.println(num1); // -128
System.out.println(num2); // 127
```
